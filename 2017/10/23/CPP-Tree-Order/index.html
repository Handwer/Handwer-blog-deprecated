C++ 二叉树的三种遍历

C++ 的二叉树有三种遍历方法，分别是前序遍历、中序遍历、后序遍历.

具体的模板

指针二叉树

//前序遍历
void PreOrder(TreeNode *root)
{
        if (root == NULL)
            return;
        Visit root->val;
        PreOrder(root->left);
        PreOrder(root->right);

}

//中序遍历
void InOrder(TreeNode *root)
{
        if (root == NULL)
            return;
        InOrder(root->left);
        Visit root->val;
        InOrder(root->right);

}


//后序遍历
void PostOrder(TreeNode *root)
{
        if (root == NULL)
            return;
        PostOrder(root->left);
        PostOrder(root->right);
        Visit root->val;

}
