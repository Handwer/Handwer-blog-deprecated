背包问题

问法
有n个物品，每个物品的重量为w[i]，每个物品的价值为v[i]。现在有一个背包，它能容纳的重量为w，问：当你面对n个有价值的物品，你的背包可以带走的最大价值是多少？
一般模板
#include <iostream>
#include<algorithm>
using namespace std;
int main() {
    int total_weight = 10;
    int w[6] = { 0,5,4,3,2,1};
    int v[6] = { 0,1,2,3,4,5};
    int dp[11] = { 0 };

    for (int i = 1; i <= 5; i++)
        for (int j = 10; j >= w[i]; j--)
            dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
    cout << dp[10] << endl;
    return 0;
}
